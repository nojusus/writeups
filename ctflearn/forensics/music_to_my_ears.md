Download the provided file "hereisyourflag.m4a" and go to https://hexed.it. Change the first 3 bytes to 00 00 00. Change the 6-12 bytes to 74 79 70 4D 34 41 20. Select the last 4 bytes in the first row, right click and "Delete selected bytes". Save the new file and run the command
```bash
audacity
```
Play the saved file and write down the spelled letters to get the flag.
