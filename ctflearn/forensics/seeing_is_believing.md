Download the provided file "message.zip" and extract it. Rename the found file "help.me" to "help.ogg". Run the command
```bash
sonic-visualiser
```
Go to "File" > "Open..." > "help.ogg". Go to "Layer" > "Add Spectrogram". Set the "Color" to "White on Black", "Normalization" to "View" and "Threshold" to "-80dB". Scan the QR code to get the flag.
